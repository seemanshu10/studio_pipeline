# Studio_pipeline

What Studio_template has :
PROJECTS → all active projects
LIBRARY → reusable assets (HDRI, rigs, textures)
PIPELINE_TOOLS → your Python scripts
TEMPLATES → folder templates
CONFIG → naming rules, settings

- work Folder where temp file and working files can be saved by artists 
- Simulate a publish button so the files are saved in publish folder structure these file can be only read not overwritten 
- Publish files shot_010_001_v001 so this is the naming structure for thhese publish files .
